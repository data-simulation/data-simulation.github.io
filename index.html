Los datos son la base de cualquier proceso de trabajo con Inteligencia artificial, aunque se le dé más importancia a los algoritmos. De nada sirve el algoritmo de TikTok si una persona no usa la aplicación y genera datos con sus likes y visualizaciones.

Transformar la realidad en datos permite generar una simulación donde los algoritmos funcionen.


Esta investigación es parte de un proceso más largo a desarrollar en un trabajo fin de máster para entender cómo afectan los datos tanto al proceso creativo como a la relación con el entorno.


El formato web o hipermedia de este artículo obliga a transformar los casos de estudio y su representación en datos, unidades de información basados en números. Todas las imágenes, textos y modelos se encuentran en un repositorio de código que puede ser descargado y modificado.

1. Obtención de datos


El requisito principal para desarrollar un algoritmo es trabajar con una buena base de datos que sea capaz de describir la realidad con precisión. Para ello es necesario traducir los fenómenos concretos del entorno al lenguaje matemático que emplean los ordenadores.


Hoy en día, casi todo lo que nos rodea genera datos, desde la geolocalización del móvil hasta las páginas que visitamos o los productos que compramos.


Los siguientes ejemplos estudian cómo se mueven las personas en el entorno físico y virtual.



Urban Data Eye

Urban Data Eye se pregunta cómo obtener datos precisos del uso del espacio urbano. Para ello aprovechan la infraestructura de cámaras existente y crean un algoritmo que registra datos que antes se estaban perdiendo de manera precisa, anónima y continua. Estos datos permiten saber por dónde se mueven o no los peatones, o qué espacios se saturan y cuáles no se usan; y sirven para obtener un diagnóstico sobre un espacio urbano determinado.

Gran San Blas

Revista Arquitectura Nº 113-114 Mayo-Junio 1968 La investigación de Mario J. Gaviria analiza el barrio de San Blas tomando datos de forma analógica mediante encuestas o conteos para entender la organización y el funcionamiento del barrio.

INE

Desde noviembre de 2019 el Instituto Nacional de Estadística realiza un estudio de medición de movilidad laboral y vacacional a partir de tecnología móvil. Durante varios días al año se registra la posición de cada móvil que existe en el país para medir los desplazamientos habituales de la población.

Cookies

Las cookies son datos almacenados en los navegadores de internet que contienen información acerca del uso que hacemos de una página web, es decir, de cómo nos movemos por la red. Los datos de seguimiento sirven para elaborar perfiles de usuario y ofrecer publicidad personalizada.

2. Espacio de soluciones posibles


El espacio de soluciones posibles es la delimitación de todas las soluciones correctas o no que podría tener un problema. El número de configuraciones posibles de un tablero de ajedrez es mayor que el número de átomos del universo.

El espacio de soluciones funciona como un gimnasio repleto de máquinas de las cuales hay unas más convenientes para desarrollar una musculatura concreta. Este entorno virtual se denomina simulación y se define por sus reglas.

Los siguientes ejemplos muestran diferentes simulaciones donde se entrenan o se podrían entrenar algoritmos.



Learning To Dress

Learning To Dress: Synthesizing Human Dressing Motion via Deep Reinforcement Learning es un entorno de simulación para sintetizar el movimiento de un cuerpo poniéndose una camiseta y entrenando un robot para que realice esa tarea. Al no existir una base de datos de gente poniéndose camisetas, se simula la gravedad, las colisiones de la tela y los músculos del cuerpo para crear esos datos.

Flexible Muscle-Based Locomotion

Flexible Muscle-Based Locomotion for Bipedal Creatures muestra cómo afectan las físicas a distintas configuraciones corporales en el proceso de aprender a caminar sobre dos extremidades. Cada criatura evoluciona en cada nueva generación adaptando su forma de moverse según lo que han aprendido.

GENERATOR

GENERATOR es una propuesta de Cedric Price para un centro cultural en un solar de Florida. Define el espacio de soluciones para la configuración de unos módulos móviles que respondía a los intereses de los usuarios. Fue un intento de llevar a la realidad la idea del Fun Palace que no se completó porque no existía la tecnología necesaria.

The Universal Constructor

Grupo de investigación de la AA de Julia and John Frazer. El modelo consiste en una placa base denominada paisaje y de una serie de células que se apilan verticalmente. Cada célula tiene un código de 8 números que la define y permite comunicarse con las células adyacentes. Es un objeto que materializa el espacio abstracto de una simulación y se utilizó para cartografiar coreografías en la notación de Laban.
